Everything you need, nothing you don't

📡

Live data

Pulls price, change % and company name directly from CNBC's quote API. Updates every 10 minutes.

🎨

Colour-coded direction

Green up-arrow, red down-arrow, grey dash for flat. Updates dynamically on every data refresh.

🖼️

Optional logo

Drop a 32×32 PNG named after the ticker into the Images\ folder to show a company logo.

🖱️

Click-to-open

Left-click any card to open the CNBC quote page for that ticker in your browser.

🐧

Open Source

Add, remove, modify. Change whatever you want.

// getting started

Install in three steps

01

Download & install

Download the .rmskin package and double-click to install via Rainmeter's skin installer. Requires Rainmeter 4.5+.

02

Choose your skin

Use an existing tile or copy a folder. Edit the .ini and change the [Variables] section to set your ticker.

03

Add a logo (optional)

Drop a 32×32 PNG named after your ticker (e.g. AAPL.png) into the @Resources\Images\ folder.
